# Manual de instalación y acceso

## Descripción del prototipo

Este proyecto es un prototipo interactivo de una aplicación web centrada en el **ODS 8: Trabajo decente y crecimiento económico**.

La aplicación incluye cuatro pantallas principales:

* **Inicio**: muestra datos, historias reales y acceso a retos.
* **Aprende**: contiene cursos, derechos laborales, lectura fácil y bolsa de empleo segura.
* **Retos**: incluye trivias, progreso, logros y retos interactivos.
* **Perfil**: permite consultar datos del usuario y activar opciones de accesibilidad.

## Requisitos

Para abrir el prototipo se necesita:

* Un navegador actualizado, como Chrome, Edge o Firefox.
* Visual Studio Code.
* La extensión **Live Server** instalada en Visual Studio Code.

## Cómo abrir el prototipo

1. Descargar o descomprimir la carpeta del proyecto.
2. Abrir la carpeta completa en Visual Studio Code.
3. Entrar en la carpeta:

```text
main-page
```

4. Abrir el archivo:

```text
index.html
```

5. Hacer clic derecho sobre el archivo y seleccionar:

```text
Open with Live Server
```

6. El prototipo se abrirá en el navegador desde la pantalla de Inicio.

## Navegación

La aplicación se navega desde la barra inferior, que aparece en todas las pantallas.

Desde ahí se puede acceder a:

* Inicio
* Aprende
* Retos
* Perfil

Se recomienda empezar siempre desde:

```text
main-page/index.html
```

## Funciones principales

El prototipo permite probar:

* Navegación entre pantallas.
* Apertura y cierre de paneles.
* Notificaciones.
* Mensajes de confirmación.
* Historias con alternativa textual accesible.
* Guía de derechos laborales.
* Versión de lectura fácil.
* Bolsa de empleo segura.
* Trivia con confirmación de respuesta.
* Ajustes de accesibilidad en Perfil.

Algunas funciones están simuladas, ya que se trata de un prototipo horizontal y no de una aplicación final conectada a una base de datos.

## Accesibilidad

El prototipo incluye varias mejoras de accesibilidad:

* Modo oscuro / alto contraste.
* Aumento del tamaño del texto.
* Iconos acompañados de texto.
* Alternativas textuales para vídeos.
* Lectura fácil.
* Contraste revisado.
* Botones claros y visibles.
* Uso de etiquetas accesibles para lectores de pantalla.

## Problemas comunes

Si los cambios no aparecen en el navegador, actualizar con:

```text
Ctrl + F5
```

Si los iconos no se ven, comprobar que hay conexión a Internet.

Si la navegación no funciona, revisar que no se hayan cambiado los nombres de las carpetas.
